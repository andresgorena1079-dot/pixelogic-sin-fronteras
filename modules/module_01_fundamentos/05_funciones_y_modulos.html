Lección 5: Funciones y Módulos

A medida que tus programas crecen, la organización se vuelve crucial. En esta lección, aprenderás a crear **funciones** para empaquetar y reutilizar bloques de código, y a usar **módulos** para dividir tu proyecto en archivos lógicos y manejables.

Tema 1: Funciones - Creando Código Reutilizable

Una **función** es un bloque de código con nombre que realiza una tarea específica. En lugar de escribir el mismo código varias veces, lo defines una vez en una función y luego la "llamas" cada vez que la necesites. Esto sigue el principio **DRY (Don't Repeat Yourself - No te Repitas)**.

Definición y Llamada de una Función

  • Definición (def): Se usa la palabra clave `def` para crear una función. Dentro de los paréntesis se definen los **parámetros** (variables que la función espera recibir).
  • Retorno (return): La palabra clave `return` se usa para que la función devuelva un resultado. Una vez que se ejecuta, la función termina.
  • Llamada: Para ejecutar la función, simplemente escribes su nombre con los **argumentos** (los valores reales para los parámetros) entre paréntesis. 
# 1. Definición de la función
def sumar(numero1, numero2):
    resultado = numero1 + numero2
    return resultado

# 2. Llamada a la función
suma_total = sumar(10, 5) # 10 y 5 son los argumentos
print(f"El resultado es: {suma_total}")

# Salida Esperada en la Consola
El resultado es: 15

Argumentos por Defecto

Puedes asignar un valor por defecto a un parámetro. Si no se proporciona un argumento para ese parámetro al llamar a la función, usará el valor por defecto, haciendo que ese argumento sea opcional.

# El parámetro 'saludo' tiene un valor por defecto
def saludar_usuario(nombre, saludo="Hola"):
    print(f"{saludo}, {nombre}!")

saludar_usuario("Ana") # Usa el saludo por defecto
saludar_usuario("Luis", "Bienvenido") # Proporciona un saludo diferente

# Salida Esperada en la Consola
Hola, Ana!
Bienvenido, Luis!

Tema 2: Módulos - Organizando tu Código

Un **módulo** es simplemente un archivo de Python (`.py`) que contiene funciones, clases y variables. Sirven para organizar el código de forma lógica y para reutilizarlo en diferentes proyectos. Python viene con una enorme **Biblioteca Estándar** llena de módulos listos para usar.

Importando Módulos

Se usa la palabra clave `import` para traer las funcionalidades de un módulo a tu script actual.

  • import modulo: Importa el módulo completo. Para usar sus funciones, debes escribir `modulo.funcion()`. 
    import random # Importa el módulo para generar números aleatorios
numero_aleatorio = random.randint(1, 10)
print(f"Número aleatorio entre 1 y 10: {numero_aleatorio}")
  • from modulo import funcion: Importa solo un elemento específico del módulo. Puedes llamarlo directamente sin el prefijo del módulo. 
    from math import sqrt # Importa solo la función de raíz cuadrada
raiz_cuadrada = sqrt(25)
print(f"La raíz cuadrada de 25 es: {raiz_cuadrada}") # Salida: 5.0

Ejemplo Integrador: Calculadora de Descuentos Modulariada

Este ejemplo práctico muestra cómo separar la lógica en un módulo y usarla desde un script principal.

Paso 1: Crear el Módulo (`calculos.py`)

Este archivo contendrá nuestra función reutilizable.

# Archivo: calculos.py
def calcular_precio_final(precio_original, porcentaje_descuento):
    monto_descuento = precio_original * (porcentaje_descuento / 100)
    precio_con_descuento = precio_original - monto_descuento
    return precio_con_descuento

Paso 2: Crear el Script Principal (`app.py`)

Este archivo importará y usará la función del módulo `calculos`.

# Archivo: app.py
from calculos import calcular_precio_final # Importamos nuestra función

precio_tv = 500
descuento_oferta = 20 # 20% de descuento

# Usamos la función importada
precio_final_tv = calcular_precio_final(precio_tv, descuento_oferta)

print("--- Tienda de Electrónica ---")
print(f"Precio original de la TV: ${precio_tv}")
print(f"Precio final con {descuento_oferta}% de descuento: ${precio_final_tv}")

# Salida Esperada en la Consola
--- Tienda de Electrónica ---
Precio original de la TV: $500
Precio final con 20% de descuento: $400.0

Ejercicio 1: Módulo de Conversiones de Temperatura

Objetivo: Crear un módulo reutilizable con funciones de conversión y usarlo desde otro script.

Instrucciones:

  1. Crea un archivo llamado `conversor.py`. Dentro, define dos funciones: `celsius_a_fahrenheit(celsius)` y `fahrenheit_a_celsius(fahrenheit)`.
    Fórmulas: `F = C * 9/5 + 32` y `C = (F - 32) * 5/9`.
  2. Crea un segundo archivo llamado `clima_app.py`.
  3. En `clima_app.py`, importa las dos funciones desde tu módulo `conversor`.
  4. Usa las funciones importadas para convertir 25 grados Celsius a Fahrenheit y 77 grados Fahrenheit a Celsius.
  5. Imprime los resultados de forma clara.
Ver Solución 
# Archivo: conversor.py
def celsius_a_fahrenheit(celsius):
    return celsius * 9/5 + 32
def fahrenheit_a_celsius(fahrenheit):
    return (fahrenheit - 32) * 5/9

# Archivo: clima_app.py
from conversor import celsius_a_fahrenheit, fahrenheit_a_celsius
temp_c = 25
temp_f = 77
print(f"{temp_c}°C son {celsius_a_fahrenheit(temp_c)}°F")
print(f"{temp_f}°F son {fahrenheit_a_celsius(temp_f)}°C")
