Lección 1: El Entorno y la Sintaxis Básica

¡Bienvenido al inicio de tu viaje con Python! Antes de escribir código, un buen programador prepara sus herramientas. En esta primera lección, configuraremos un entorno de desarrollo profesional y escribiremos nuestro primer programa, entendiendo los conceptos más básicos de la sintaxis de Python.

Tema 1: Configuración del Entorno de Desarrollo

Al igual que un chef necesita una cocina bien equipada, un programador necesita un entorno de desarrollo robusto. Esto consiste en dos componentes clave: el intérprete de Python (el "motor") y un editor de código (nuestro "taller").

1.1: Instalación de Python 3.10+

El intérprete de Python es el programa que lee, traduce y ejecuta tu código. Sin él, tu computadora no podría entender las instrucciones escritas en lenguaje Python.

  • Paso 1: Ve al sitio web oficial de Python: python.org/downloads.
  • Paso 2: Descarga la última versión estable para tu sistema operativo.
  • Paso 3: Ejecuta el instalador.

⚠️ Atención (Windows): Durante la instalación, es vital marcar la casilla que dice "Add Python to PATH". Esto te permitirá ejecutar Python desde cualquier terminal sin problemas.

1.3: Uso de Entornos Virtuales (Buena Práctica)

Un entorno virtual es una carpeta aislada que contiene una instalación de Python específica para un proyecto. Esto evita que los paquetes de diferentes proyectos entren en conflicto.

¿Por qué es importante? Imagina que el Proyecto A necesita la versión 1.0 de una librería, pero el Proyecto B necesita la versión 2.0. Sin entornos virtuales, tendrías un conflicto. Con ellos, cada proyecto tiene su propio espacio aislado.

# 1. Navega a la carpeta de tu proyecto
# 2. Crea un entorno llamado "venv"
python -m venv venv

# 3. Activa el entorno
# En Windows:
.\venv\Scripts\activate
# En macOS/Linux:
source venv/bin/activate

Sabrás que está activado porque el nombre `(venv)` aparecerá al inicio de la línea de tu terminal.

Tema 2: Primer Programa y Comentarios

Ahora que tenemos el entorno listo, vamos a escribir nuestro primer programa, una tradición conocida como "Hola, Mundo".

🔵 Ficha Técnica: Función print()

Categoría: Función Incorporada (Built-in)

Sintaxis: print(objeto1, objeto2, ..., sep=' ', end='\n')

Descripción: Imprime los objetos especificados en la consola. Es la herramienta fundamental para mostrar información al usuario y para depurar (verificar) el estado de las variables.

Ejemplo Práctico:

# Código de Entrada
nombre = "Mundo"
print("Hola,", nombre, "!")

# Salida Esperada en la Consola
Hola, Mundo !

Ejercicio 1: Mensaje de Bienvenida (Conceptual)

Objetivo: Practicar el uso de la función print() con múltiples argumentos y el parámetro sep.

Instrucciones:

  3. Utiliza el parámetro sep=" | " para que los datos aparezcan separados por una barra vertical.
  4. Usa el botón "Ejecutar" en la zona de práctica para ver el resultado.
Ver Solución
