Lección 3: Operadores en Python

Los programas no solo guardan datos; los manipulan, comparan y operan con ellos. Los **operadores** son los símbolos especiales que le dicen a Python que realice una operación. En esta lección, exploraremos las herramientas que nos permiten dar vida y lógica a nuestras variables.

Tema 1: Operadores Aritméticos

Son los operadores que usas para realizar cálculos matemáticos. Te permiten ejecutar desde sumas simples hasta operaciones más complejas como el cálculo de potencias o el resto de una división, siendo la base de cualquier manipulación numérica en tu código.

Operadores Aritméticos Comunes

  • Suma (+): Suma dos números. 
    print(f"10 + 5 = {10 + 5}") # Salida: 15
  • Resta (-): Resta un número de otro. 
    print(f"20 - 8 = {20 - 8}") # Salida: 12
  • Multiplicación (*): Multiplica dos números. 
    print(f"7 * 6 = {7 * 6}") # Salida: 42
  • División (/): Divide. El resultado siempre es un `float`. 
    print(f"10 / 4 = {10 / 4}") # Salida: 2.5
  • División Entera (//): Divide y redondea hacia abajo al entero más cercano. 
    print(f"10 // 4 = {10 // 4}") # Salida: 2
  • Módulo (%): Devuelve el resto de una división entera. 
    print(f"El resto de 10 / 4 es: {10 % 4}") # Salida: 2
  • Exponente (**): Eleva un número a la potencia de otro. 
    print(f"2 elevado a la 3 es: {2 ** 3}") # Salida: 8

Tema 2: Operadores de Asignación

Estos operadores se utilizan para asignar o actualizar el valor de una variable. El más común es `=`, pero existen versiones combinadas que actúan como atajos para modificar una variable basándose en su valor actual, haciendo el código más conciso.

Operadores de Asignación Comunes

  • Asignación (=): Asigna el valor de la derecha a la variable de la izquierda.
  • Asignación Compuesta (+=, -=, *=, etc.): Realiza una operación y asigna el resultado a la misma variable. Es un atajo muy común. 
    puntos = 100
puntos += 20  # Es lo mismo que: puntos = puntos + 20
print(f"Puntos totales: {puntos}") # Salida: 120

Tema 3: Operadores de Comparación

Comparan dos valores y el resultado de la expresión es siempre un valor booleano: `True` o `False`. Son la base fundamental para la toma de decisiones y el control de flujo en cualquier programa.

Operadores de Comparación Comunes

  • Igual a (==): Comprueba si dos valores son iguales. 
    print(f"¿10 es igual a 10?: {(10 == 10)}") # Salida: True
  • No igual a (!=): Comprueba si dos valores son diferentes. 
    print(f"¿'hola' es diferente de 'adios'?: {('hola' != 'adios')}") # Salida: True
  • Mayor que (>): Comprueba si el valor de la izquierda es mayor que el de la derecha. 
    print(f"¿20 es mayor que 10?: {(20 > 10)}") # Salida: True
  • Menor o igual que (<=): Comprueba si el valor de la izquierda es menor o igual que el de la derecha. 
    print(f"¿15 es menor o igual que 15?: {(15 <= 15)}") # Salida: True

Tema 4: Operadores Lógicos

Estos operadores te permiten combinar múltiples expresiones booleanas para crear condiciones más complejas y potentes. Son esenciales para construir una lógica de decisión que dependa de varios factores.

Operadores Lógicos Comunes

  • Y Lógico (and): Devuelve `True` solo si **ambas** condiciones son verdaderas. 
    puede_pasar = (12 >= 10) and (150 > 140)
print(f"¿Puede pasar?: {puede_pasar}") # Salida: True
  • O Lógico (or): Devuelve `True` si **al menos una** de las condiciones es verdadera. 
    tiene_descuento = (True or False)
print(f"¿Tiene descuento?: {tiene_descuento}") # Salida: True
  • Negación Lógica (not): Invierte el valor booleano de una condición. 
    esta_lloviendo = False
hay_sol = not esta_lloviendo
print(f"¿Hay sol?: {hay_sol}") # Salida: True

Ejemplo Integrador: Lógica de un Videojuego

Este mini-programa funcional utiliza todos los operadores de la lección para analizar dos números y mostrar un reporte completo.

# --- INICIO DEL SCRIPT ---
num1 = 20; num2 = 5
print(f"--- Analizando los números: {num1} y {num2} ---\n")

# Tema 1: Operadores Aritméticos
print("** Operaciones Aritméticas **")
print(f"Suma: {num1 + num2}")
print(f"Resta: {num1 - num2}")
print(f"Multiplicación: {num1 * num2}")
print(f"División: {num1 / num2}")
print(f"División Entera: {num1 // num2}")
print(f"Módulo: {num1 % num2}")
print(f"Exponente (num2 ** 2): {num2 ** 2}\n")

# Tema 2: Operadores de Comparación
print("** Operaciones de Comparación **")
print(f"¿Son iguales? (==): {num1 == num2}")
print(f"¿Son diferentes? (!=): {num1 != num2}")
print(f"¿num1 > num2?: {num1 > num2}")
print(f"¿num1 <= num2?: {num1 <= num2}\n")

# Tema 3: Operadores Lógicos
print("** Operaciones Lógicas **")
condicion1 = num1 > 10  # True
condicion2 = num2 < 3   # False
print(f"¿condicion1 Y condicion2? (and): {condicion1 and condicion2}")
print(f"¿condicion1 O condicion2? (or): {condicion1 or condicion2}")
print(f"¿NO condicion2? (not): {not condicion2}\n")

# Tema 4: Operadores de Asignación
print("** Operaciones de Asignación **")
resultado = 100
resultado -= 50  # resultado ahora es 50
resultado *= 3   # resultado ahora es 150
print(f"Resultado final de asignación compuesta: {resultado}")

Ejercicio 1: Sistema de Puntuación de Gimnasia

Objetivo: Crear un "mini-programa" que calcule la puntuación final de un gimnasta y determine si califica para una medalla, usando múltiples operadores.

Instrucciones:

  1. Declara las puntuaciones de 3 jueces (ej: `juez1 = 9.5`, `juez2 = 9.8`, `juez3 = 9.2`).
  2. Calcula la `puntuacion_total` sumando las tres puntuaciones.
  3. Calcula la `puntuacion_promedio` dividiendo el total entre 3.
  4. Declara una variable `puntuacion_minima_medalla` con valor `9.5`.
  5. Crea una variable booleana `califica_para_medalla` que sea `True` si el promedio es mayor o igual a la puntuación mínima.
  6. Imprime la puntuación de cada juez, la puntuación promedio final y un mensaje que diga "Califica para medalla: [True/False]".
Ver Solución 
juez1, juez2, juez3 = 9.5, 9.8, 9.2
puntuacion_total = juez1 + juez2 + juez3
puntuacion_promedio = puntuacion_total / 3
puntuacion_minima_medalla = 9.5
califica_para_medalla = puntuacion_promedio >= puntuacion_minima_medalla

print("--- Evaluación de Gimnasia ---")
print(f"Puntuaciones: {juez1}, {juez2}, {juez3}")
print(f"Promedio Final: {puntuacion_promedio}")
print("-----------------------------")
print(f"¿Califica para medalla?: {califica_para_medalla}")

Ejercicio 2: Simulador de Compra con Cupón

Objetivo: Usar operadores lógicos para aplicar un descuento solo si se cumplen varias condiciones.

Instrucciones:

  1. Declara las variables `total_compra` (float), `es_cliente_vip` (bool) y `tiene_cupon` (bool).
  2. Un descuento del 15% se aplica si el `total_compra` es mayor a 100 **Y** (el cliente `es_cliente_vip` **O** `tiene_cupon`).
  3. Crea una variable booleana `aplica_descuento` que contenga el resultado de esa lógica.
  4. Usa un `if` para imprimir el precio final: si `aplica_descuento` es `True`, calcula y muestra el precio con descuento; si no, muestra el precio original.
Ver Solución 
total_compra = 120.0
es_cliente_vip = False
tiene_cupon = True

aplica_descuento = total_compra > 100 and (es_cliente_vip or tiene_cupon)

print("--- Resumen de Compra ---")
if aplica_descuento:
    precio_final = total_compra * 0.85 # 15% de descuento
    print("¡Descuento aplicado!")
    print(f"Total a pagar: ${precio_final}")
else:
    print("No se aplicó ningún descuento.")
    print(f"Total a pagar: ${total_compra}")
