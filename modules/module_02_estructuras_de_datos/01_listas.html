Lección 1: Listas

Una **lista** es la estructura de datos más fundamental de Python. Es una colección **ordenada** de elementos (cada uno con una posición o índice) y **mutable** (puedes cambiar su contenido en cualquier momento).

Tema 1: Indexado y Acceso a Elementos

Cada elemento en una lista tiene una posición específica, identificada por un número llamado **índice**.

Indexado Positivo (desde el principio)

El conteo de los índices en Python siempre empieza en **cero**.

Índice0123
Elemento"Ana""Luis""Marta""Juan"
jugadores = ["Ana", "Luis", "Marta", "Juan"]
print(f"El jugador en el índice 0 es: {jugadores[0]}") # Salida: Ana

Indexado Negativo (desde el final)

Permite acceder a los elementos empezando desde el final. El último elemento es el índice `-1`.

Elemento"Ana""Luis""Marta""Juan"
Índice-4-3-2-1
jugadores = ["Ana", "Luis", "Marta", "Juan"]
print(f"El último jugador es: {jugadores[-1]}") # Salida: Juan

Tema 2: Métodos de Listas

Las listas vienen con métodos para manipular su contenido.

🔵 Ficha Técnica: list.append(elemento)

Descripción: Añade un único elemento al **final** de la lista.

tareas = ["Limpiar"]
tareas.append("Estudiar")
print(tareas) # Salida: ['Limpiar', 'Estudiar']

🔵 Ficha Técnica: list.extend(iterable)

Descripción: Une una lista con otro iterable (como otra lista), añadiendo cada elemento al final.

lista_a = [1, 2]; lista_b = [3, 4]
lista_a.extend(lista_b)
print(lista_a) # Salida: [1, 2, 3, 4]

🔵 Ficha Técnica: list.insert(indice, elemento)

Descripción: Inserta un elemento en una posición específica, desplazando el resto.

ranking = ["Oro", "Bronce"]
ranking.insert(1, "Plata")
print(ranking) # Salida: ['Oro', 'Plata', 'Bronce']

🔵 Ficha Técnica: list.remove(valor)

Descripción: Elimina la **primera ocurrencia** de un valor especificado.

frutas = ['manzana', 'pera', 'manzana']
frutas.remove('manzana')
print(frutas) # Salida: ['pera', 'manzana']

🔵 Ficha Técnica: list.pop(indice)

Descripción: Elimina y **retorna** el elemento de la posición especificada. Si se omite el índice, elimina el último.

baraja = ["As", "Rey", "Reina"]
carta_tomada = baraja.pop()
print(f"Carta: {carta_tomada}, Baraja: {baraja}") # Salida: Carta: Reina, Baraja: ['As', 'Rey']

🔵 Ficha Técnica: list.sort()

Descripción: Ordena los elementos de la lista. **Modifica la lista original**.

puntuaciones = [95, 88, 100]
puntuaciones.sort()
print(puntuaciones) # Salida: [88, 95, 100]

🔵 Ficha Técnica: list.reverse()

Descripción: Invierte el orden de los elementos. **Modifica la lista original**.

numeros = [1, 2, 3, 4]
numeros.reverse()
print(numeros) # Salida: [4, 3, 2, 1]

Ejemplo Integrador: Gestión de una Playlist Musical

Este script funcional aplica todos los conceptos de la lección para gestionar una lista que representa una playlist.

playlist = ["Bohemian Rhapsody", "Stairway to Heaven", "Hotel California"]
print(f"Playlist inicial: {playlist}\n")

# Añadir canciones
playlist.append("Imagine")
playlist.insert(1, "Like a Rolling Stone")
print(f"Playlist actualizada: {playlist}\n")

# Eliminar una canción por nombre y otra por posición
playlist.remove("Stairway to Heaven")
cancion_eliminada = playlist.pop(0)
print(f"Se eliminó '{cancion_eliminada}'.")
print(f"Playlist después de eliminar: {playlist}\n")

# Buscar y contar
posicion = playlist.index("Hotel California")
print(f"'Hotel California' está en la posición: {posicion}")
playlist.append("Imagine")
veces_imagine = playlist.count("Imagine")
print(f"'Imagine' aparece {veces_imagine} veces.\n")

# Ordenar y revertir
playlist.sort()
print(f"Playlist ordenada: {playlist}")
playlist.reverse()
print(f"Playlist invertida: {playlist}")
