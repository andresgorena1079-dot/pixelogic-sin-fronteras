Lección 2: Tuplas y Conjuntos (Sets)

Además de las listas, Python ofrece otras estructuras de datos para situaciones específicas. En esta lección, exploraremos las **tuplas**, que son como "listas inmutables", y los **conjuntos (sets)**, colecciones de elementos únicos sin un orden específico.

Tema 1: Tuplas - Colecciones Inmutables

Una **tupla** es muy similar a una lista (es ordenada y se accede por índice), pero con una diferencia crucial: es **inmutable**. Una vez que creas una tupla, no puedes añadir, eliminar o modificar sus elementos. Piensa en ellas como una "caja de cristal" sellada, perfecta para datos que no deben cambiar.

Creación, Acceso y Desempaquetado

Se crean con paréntesis `()` y se accede a sus elementos con índices. Una de sus características más potentes es el "desempaquetado", que permite asignar sus elementos a variables individuales.

# Código de Entrada
configuracion_db = ("localhost", 5432, "admin")
# Desempaquetado de tuplas
host, puerto, usuario = configuracion_db
print(f"Conectando a {host} en el puerto {puerto}")

# Salida Esperada en la Consola
Conectando a localhost en el puerto 5432

Métodos de Tuplas

  • .count(valor): Cuenta cuántas veces aparece un `valor` en la tupla. 
    notas = (10, 8, 9, 10, 7, 10)
veces_diez = notas.count(10)
print(f"El número 10 aparece {veces_diez} veces.") # Salida: 3
  • .index(valor): Devuelve el índice de la primera ocurrencia de un `valor`. 
    dias_semana = ("Lunes", "Martes", "Miércoles")
posicion_martes = dias_semana.index("Martes")
print(f"Martes está en el índice: {posicion_martes}") # Salida: 1

Tema 2: Conjuntos (Sets) - Colecciones de Elementos Únicos

Un **conjunto (set)** es una colección que tiene dos características clave: **no tiene un orden** específico y **no permite elementos duplicados**. Su principal superpoder es realizar operaciones matemáticas de conjuntos de forma muy eficiente.

Métodos de Modificación

  • .add(elemento): Añade un elemento. Si ya existe, no hace nada. 
    ingredientes = {"tomate", "cebolla"}
ingredientes.add("ajo")
print(ingredientes) # Salida (el orden puede variar): {'cebolla', 'ajo', 'tomate'}
  • .remove(elemento): Elimina un elemento. Lanza `KeyError` si el elemento no existe. 
    ingredientes = {"tomate", "cebolla"}
ingredientes.remove("tomate")
print(ingredientes) # Salida: {'cebolla'}
  • .discard(elemento): Elimina un elemento. Si no existe, **no hace nada** (es más seguro que `remove`). 
    ingredientes = {"tomate", "cebolla"}
ingredientes.discard("pimiento") # No da error
print(ingredientes) # Salida: {'cebolla', 'tomate'}
  • .pop(): Elimina y retorna un elemento aleatorio del conjunto. 
    tags = {"python", "web", "dev"}
tag_eliminado = tags.pop()
print(f"Tag eliminado: {tag_eliminado}")
print(f"Tags restantes: {tags}")

Operaciones de Conjuntos

  • Unión (|): Combina dos sets, devolviendo todos los elementos únicos de ambos. 
    set_a = {1, 2, 3}; set_b = {3, 4, 5}
union = set_a | set_b
print(union) # Salida: {1, 2, 3, 4, 5}
  • Intersección (&): Devuelve solo los elementos que están presentes en ambos sets. 
    set_a = {1, 2, 3}; set_b = {3, 4, 5}
interseccion = set_a & set_b
print(interseccion) # Salida: {3}
  • Diferencia (-): Devuelve los elementos que están en el primer set pero no en el segundo. 
    set_a = {1, 2, 3}; set_b = {3, 4, 5}
diferencia = set_a - set_b
print(diferencia) # Salida: {1, 2}

Ejemplo Integrador: Análisis de Datos de Encuestas

Este script utiliza tuplas para almacenar respuestas (que no deben cambiar) y conjuntos para analizarlas (encontrar respuestas únicas, comparar grupos, etc.), integrando todos los conceptos de la lección.

# 1. Usamos una LISTA de TUPLAS para almacenar datos inmutables.
# Cada tupla es (id_usuario, respuesta), un registro que no debe cambiar.
respuestas_encuesta = [
    (101, "Python"),
    (102, "Java"),
    (103, "Python"),
    (104, "C++"),
    (105, "Java")
]

# 2. Usamos un CONJUNTO para obtener todas las respuestas ÚNICAS.
# El método .add() se encarga de ignorar los duplicados automáticamente.
lenguajes_mencionados = set()
for id_usuario, lenguaje in respuestas_encuesta: # Desempaquetado de tuplas
    lenguajes_mencionados.add(lenguaje)

print(f"Lenguajes únicos mencionados: {lenguajes_mencionados}\n")

# 3. Usamos OPERACIONES DE CONJUNTOS para analizar los resultados.
lenguajes_web = {"Python", "Java", "JavaScript"}
lenguajes_compilados = {"C++", "Java", "Rust"}

comunes_web = lenguajes_mencionados & lenguajes_web
no_compilados = lenguajes_mencionados - lenguajes_compilados

print("--- Análisis de la Encuesta ---")
print(f"Lenguajes de la encuesta que también son de desarrollo web: {comunes_web}")
print(f"Lenguajes de la encuesta que NO son compilados: {no_compilados}")

Ejercicio 1: Gestión de Coordenadas Geográficas

Objetivo: Usar una lista de tuplas para almacenar datos fijos y practicar el acceso y desempaquetado.

Instrucciones:

  1. Crea una lista de tuplas llamada `ciudades`, donde cada tupla contenga el nombre de una ciudad y sus coordenadas (lat, lon).
  2. Usa un bucle `for` y el **desempaquetado de tuplas** para imprimir un mensaje formateado para cada ciudad, como: `"Ciudad: [Nombre], Lat: [Latitud], Lon: [Longitud]"`.
  3. Usa el método `.count()` en la lista para ver si una ciudad específica (como una tupla completa) está duplicada.
Ver Solución 
ciudades = [
    ("Nueva York", 40.7128, -74.0060),
    ("Tokio", 35.6895, 139.6917),
    ("Londres", 51.5074, -0.1278)
]
print("--- Coordenadas de Ciudades ---")
for nombre, lat, lon in ciudades:
    print(f"Ciudad: {nombre}, Lat: {lat}, Lon: {lon}")

Ejercicio 2: Comparación de Habilidades de Equipos

Objetivo: Usar conjuntos y sus operaciones para analizar las habilidades de dos equipos.

Instrucciones:

  1. Define dos conjuntos, `habilidades_dev` y `habilidades_design`, cada uno con varias habilidades (strings).
  2. Usa la operación de **intersección** (`&`) para encontrar las `habilidades_comunes`.
  3. Usa la operación de **unión** (`|`) para encontrar el `conjunto_total_habilidades`.
  4. Usa la operación de **diferencia** (`-`) para encontrar las `habilidades_solo_dev`.
  5. Imprime los tres conjuntos resultantes con mensajes claros.
Ver Solución 
habilidades_dev = {"Python", "SQL", "Git", "API"}
habilidades_design = {"Figma", "UX", "Git", "HTML"}

habilidades_comunes = habilidades_dev & habilidades_design
conjunto_total_habilidades = habilidades_dev | habilidades_design
habilidades_solo_dev = habilidades_dev - habilidades_design

print(f"Habilidades en común: {habilidades_comunes}")
print(f"Todas las habilidades del equipo: {conjunto_total_habilidades}")
print(f"Habilidades que solo tienen los desarrolladores: {habilidades_solo_dev}")
