Lecci√≥n 2: Expresiones Regulares

Las **expresiones regulares** (o "regex") son un "mini-lenguaje" especializado en la b√∫squeda y manipulaci√≥n de patrones en texto. Son una herramienta fundamental para tareas como validar formatos (emails, n√∫meros de tel√©fono), encontrar datos espec√≠ficos en un texto largo o reemplazar partes de una cadena de caracteres. En Python, trabajamos con ellas a trav√©s del m√≥dulo re.

Tema 1: Funciones B√°sicas del M√≥dulo re

El m√≥dulo `re` nos ofrece varias funciones para trabajar con patrones. Las m√°s comunes son `search`, `findall` y `sub`.

üîµ Ficha T√©cnica: re.search(patron, texto)

Descripci√≥n: Escanea el `texto` buscando la **primera ubicaci√≥n** donde el `patron` produce una coincidencia.

Retorna: Un objeto "match" si encuentra una coincidencia, o `None` si no encuentra nada.

Ejemplo Pr√°ctico:

import re
texto = "El n√∫mero de tel√©fono es 123-456-7890."
patron = r"\d{3}-\d{3}-\d{4}" # \d representa un d√≠gito, {3} significa 3 veces

coincidencia = re.search(patron, texto)
if coincidencia:
    print(f"N√∫mero encontrado: {coincidencia.group(0)}")
else:
    print("No se encontr√≥ un n√∫mero.")

# Salida Esperada en la Consola
N√∫mero encontrado: 123-456-7890

üîµ Ficha T√©cnica: re.findall(patron, texto)

Descripci√≥n: Encuentra **todas** las subcadenas que coinciden con el `patron` y las devuelve como una lista de strings.

Retorna: Una lista de strings.

Ejemplo Pr√°ctico:

import re
texto = "Los correos son user1@mail.com y user2@domain.org"
patron_email = r"\w+@\w+\.\w+" # \w+ significa "una o m√°s letras/n√∫meros"

emails = re.findall(patron_email, texto)
print(f"Emails encontrados: {emails}")

# Salida Esperada en la Consola
Emails encontrados: ['user1@mail.com', 'user2@domain.org']

üîµ Ficha T√©cnica: re.sub(patron, reemplazo, texto)

Descripci√≥n: Busca todas las ocurrencias del `patron` en el `texto` y las reemplaza con el string de `reemplazo`.

Retorna: Un nuevo string con los reemplazos hechos.

Ejemplo Pr√°ctico:

import re
texto = "Mi n√∫mero es 12345 y el tuyo 54321"
texto_censurado = re.sub(r"\d", "X", texto) # Reemplaza cada d√≠gito (\d) por una 'X'

print(texto_censurado)

# Salida Esperada en la Consola
Mi n√∫mero es XXXXX y el tuyo XXXXX
